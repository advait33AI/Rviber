# rviber 0.1.0

* Initial release
* Chat panel inside RStudio (viewer pane or floating dialog)
* 8 registered addins: Chat, Explain, Fix, Improve, Complete, Generate, Plot, Run+Debug
* Providers: Groq, Google Gemini, OpenAI, Anthropic, Mistral, Together AI, Ollama, Custom
* Interactive setup wizard (`rviber_setup()`)
* Persistent config via `~/.rviber/config.json`
* All features callable as plain R functions
