# Keyboard Shortcuts

## Built-in shortcuts in the panel

| Action | Shortcut |
|--------|----------|
| Send chat message | Ctrl+Enter (Windows/Linux) / Cmd+Enter (Mac) |

---

## Set custom RStudio keyboard shortcuts

You can bind any rviber addin to a keyboard shortcut in RStudio.

**How to set a shortcut:**

1. Go to **Tools → Modify Keyboard Shortcuts**
2. Search for **rviber** in the search box
3. Click in the Shortcut column next to the addin you want
4. Press your desired key combination
5. Click **Apply**

**Recommended shortcuts:**

| Addin | Suggested shortcut |
|-------|--------------------|
| rviber: Open AI Assistant | Ctrl+Shift+A / Cmd+Shift+A |
| rviber: Explain Selected Code | Ctrl+Shift+E / Cmd+Shift+E |
| rviber: Fix Bugs in Selection | Ctrl+Shift+F / Cmd+Shift+F |
| rviber: Complete My Code | Ctrl+Shift+K / Cmd+Shift+K |
| rviber: Generate Code | Ctrl+Shift+G / Cmd+Shift+G |

> Check that your chosen shortcut doesn't conflict with an existing RStudio shortcut. You can see all shortcuts at **Tools → Keyboard Shortcuts Help**.

## All registered addins

These all appear in **Addins → rviber**:

| Addin name | What it does |
|------------|-------------|
| rviber: Open AI Assistant | Opens the full panel |
| rviber: Explain Selected Code | Explains selected code |
| rviber: Fix Bugs in Selection | Fixes and replaces selected code |
| rviber: Improve Selected Code | Refactors selected code |
| rviber: Complete My Code | Autocompletes from cursor |
| rviber: Generate Code from Description | Opens input dialog, generates code |
| rviber: Generate ggplot2 Chart | Opens input dialog, generates chart code |
| rviber: Setup / Change Provider | Runs the setup wizard |
